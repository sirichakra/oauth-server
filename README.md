# OAuth Authorization Server (Node.js)

## 🚀 Overview
This project implements a simplified OAuth 2.0 Authorization Server with:

- Authorization Code Flow
- PKCE (Proof Key for Code Exchange)
- JWT Access Tokens
- Refresh Tokens
- Protected Routes
- PostgreSQL Database
- Dockerized setup

---

## 🛠 Tech Stack
- Node.js (Express)
- PostgreSQL
- JWT (jsonwebtoken)
- Docker
